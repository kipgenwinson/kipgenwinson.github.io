How to publish

  1. Download each photo above and drop the files into your repo’s assets/updates/ folder.
  2. Create a new file in _updates/ using the filename shown, and paste the generated contents into it.
  3. Commit & push. GitHub Pages rebuilds and the post appears at the top of Field Notes.
